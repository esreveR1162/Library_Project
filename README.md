# Система управления личной библиотекой

Консольное приложение для управления библиотекой книг, разработанное на C++ с использованием ООП.

## Требования
- C++17
- CMake 3.10+

## Сборка и запуск
```bash
mkdir build
cd build
cmake ..
make
./library
```
## Функционал 
```
- Управление книгами
- Управление пользователями
- Выдача и возврат книг
- Сохранение данных в файл
```
